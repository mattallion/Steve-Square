# Steve-Square
Video game made using Unity and C#.

Steps to Play:
  1. Download the Build zip file.
  2. After downloading, unzip and run the Steve Square.exe file.
  3. Play the game!
  
Refer to the Scripts folder to check out the coding behind the game :)
